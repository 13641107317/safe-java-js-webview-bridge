Java Interface 可用接口列表测试

  • 确认框弹出
    代码：
    HostApp.alert('我是确认框');

  • 气泡提示
    代码：
    HostApp.toast('我是气泡');

  • 测试代码执行时间
    代码：
    HostApp.testLossTime(new Date().getTime() + '');

  • 获得设备IMSI
    代码：
    HostApp.alert(HostApp.getIMSI());

  • 是否存在纵向滚动条
    代码：
    HostApp.alert(HostApp.existVerticalScrollbar());

  • 传入JSON对象,返回对象的一个键值对
    代码：
    HostApp.alert(HostApp.passJson2Java({'key1': 'first-value', 'key2': 'secd-value'}));

  • 传入JSON,返回相同的JSON对象
    代码：
    HostApp.alert( HostApp.retBackPassJson({'b1': 'back-value1', 'b2': 'back-value2'}) );
